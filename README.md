# MineMap

マインクラフトの座標情報を管理するWebアプリです。

## Frameworks + Modules

- FuelPHP 1.8.2
- Materialize 1.0.0
- Vue.js 2.6.10
- Laravel Mix 4.1.2

## init

```sh
$ cd minecraft-map
$ composer install
$ npm install
$ docker-compose build
```

## local server

`http://localhost:8080/`

```sh
$ npm run dev
$ docker-compose up -d
```
